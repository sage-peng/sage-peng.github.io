## 删除链表元素
- 删除元素的时候如果有可能删除head，要准备一个占位的dummy指向头部，返回的时候返回dummy->next防止找不到链表的head；
### LeetCode237. 删除链表中的节点（ 腾讯）
[LeetCode237. 删除链表中的节点](https://leetcode.cn/problems/delete-node-in-a-linked-list/)

- tips：
    - 把下一个节点的值赋值给自己


```python
class Solution(object):
    def deleteNode(self, node):
        node.val=node.next.val
        node.next=node.next.next
```

```c
class Solution
{
public:
    void deleteNode(ListNode *node)
    {
        node->val = node->next->val;
        node->next = node->next->next;
    }
};
```

### LeetCode19. 删除链表的倒数第 N 个结点（ ⭐️⭐️⭐️高频、剑指II、HOT100）
[LeetCode19. 删除链表的倒数第 N 个结点.](https://leetcode.cn/problems/remove-nth-node-from-end-of-list/)

- tips：
    - 快慢指针
    - 注意占位指针的使用，防止删的是头部


```python
class Solution:
    def removeNthFromEnd(self,head,n):
        dummy=ListNode(-1,head)
        slow,fast=dummy,dummy
        for i in range(n):
            fast=fast.next
        while(fast.next):
            slow=slow.next
            fast=fast.next
        slow.next=slow.next.next
        return dummy.next
```

```c
class Solution
{
public:
    ListNode* removeNthFromEnd(ListNode *head,int n) {
        ListNode* dummy = new ListNode(-1, head);
        ListNode* slow = dummy;
        ListNode* fast = dummy;

        while (n)
        {
            fast = fast->next;
            n--;
        }

        while (fast->next)
        {
            fast = fast->next;
            slow = slow->next;
        }
        slow->next = slow->next->next;
        return dummy->next;

    }
};
```

### LeetCode83. 删除排序链表中的重复元素 I

- tips：

```c
class Solution
{
public:
    ListNode* deleteDuplicates(ListNode* head) {
        if (!head)
        {
            return NULL;

        }
        ListNode* slow = head;
        ListNode* fast = head;

        while (fast)
        {
            if (slow->val!=fast->val)
            {
                //紧挨着时不用连接，浪费了效率
                slow->next = fast;
                slow = fast;
            }

            fast = fast->next;
        }
        slow->next = fast;
        return head;
    }
};
```

### LeetCode82. 删除排序链表中的重复元素 II（⭐️⭐️高频、剑指II）
[LeetCode82. 删除排序链表中的重复元素.](https://leetcode.cn/problems/remove-duplicates-from-sorted-list-ii/)

- 对比方法一和方法二:
    - 方法一是可以移动并且不等条件成立才判断,快结束时不判断
    - 方法二除了相等每次都判断,包含快结束
- 应该是除了相等之外每次都判断

- 注意：
    - slow指向的节点一定会被接受（如果不接受他，就无法删了它，要删他，就要在他的前面）
    - slow 有两个操作：
        - 改变slow->next：在slow和slow->next中山的节点全部被删除
        - 改变slow:表示这个节点被接受了（如果不接受他，就无法删了它，要删他，就要在他的前面）

```python
#method1
dummy=ListNode(-1,head)
        pre , cur=dummy,head

            return

            if cur.next and cur.val!=cur.next.val:

                    pre.next=cur.next

        if pre.next.next != cur:

class Solution:
    #头结点有可能被删除,这是要创建一个节点指向头结点,同时用快指针自己和next比较,pre只记录不操作
    def deleteDuplicates(self, head) :

        pre , cur=dummy,head

            return

```

```c
class Solution
{
public:
    ListNode* deleteDuplicates(ListNode* head)
    {
        ListNode* dummy = new ListNode(-1, head);
        ListNode* slow = dummy;
        ListNode* fast = head;

        while (fast)
        {
            //去重：来到重复的最后一个
            while (fast->next && fast->val == fast->next->val)fast = fast->next;
             //这里的slow在fast前面；所以循环完毕不要在往前一步，否则slow不能在fast前面
            if (slow->next!=fast)//必须就是我的下一个才可以接受
            {
                slow->next = fast->next;//指过去，表示中间的都作废了
            }
            else {
                slow = fast;//移过去，表示前面的已经接受了
            }
            fast = fast->next;
        }
        
        return dummy->next;
    }
};
```

### LeetCode25. K 个一组翻转链表（难）（⭐️高频）
[LeetCode25. K 个一组翻转链表.](https://leetcode.cn/problems/reverse-nodes-in-k-group/)

```c
class Solution {
public:
    ListNode* reverse(ListNode* head,ListNode* end) {
       
            ListNode* pre = NULL,*cur=head;
                
            while (cur!=end)
            {
                ListNode* temp = cur->next;
                cur->next = pre;
                pre = cur;
                cur = temp;
            }
            return pre;
        }
    ListNode* reverseKGroup(ListNode* head, int k) {
        if (!head || !head->next) {
                return head;
        }

        ListNode* dummy = new ListNode(-1);
        dummy->next = head;
        ListNode* rear = dummy;
        ListNode* k_star = head, * k_end = head;
        while (k_end) {
            for (int i = 0; i < k; i++)
            {
                if (!k_end) {
                    rear->next = k_star;
                    return dummy->next;
                }
                k_end = k_end->next;
            }
            rear->next = reverse(k_star, k_end);
            rear = k_star;
            k_star = k_end;
        }
        return dummy->next;
    }
};
```
